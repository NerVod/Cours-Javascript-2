Élément impossible à sélectionner sur tous les navigateurs Internet

Élément sélectionnable.

L'attribut unselectable est nécessaire sur Opéra et IE < 10. La classe CSS ne fonctionne pas sur ces navigateurs.

Attention les sous-éléments n'héritent pas de l'attribut unselectable sur Opéra et IE < 10. Par exemple :

Pas séléctionnable sur Opéra et IE < 10. Pas séléctionnable sur les autres navigateurs Internet...

Séléctionnable sur Opéra et IE < 10 ! Pas séléctionnable sur les autres navigateurs Internet...

Si on a de nombreux sous-éléments, il faut créer un code JS qui place l'attribut unselectable sur tous les enfants pour gérer la compatibilité avec Opéra et IE < 10.

Voir : http://stackoverflow.com/questions/2326004/prevent-selection-in-html
